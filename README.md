# Render Engine

A work in progress game engine written in Rust.

# Sample Scene
![image](https://user-images.githubusercontent.com/36389791/157958692-ff50feee-ea31-4355-9019-88ef7695fee3.png)

# Features

* Multithreaded pipeline to quickly process and render entities
* Deferred shading to allow for rendering many lights
* Entity component system for flexible entity properties
* Shadow mapping for numerous light sources

Technologies Used

* Rust
* OpenGL 4.5
